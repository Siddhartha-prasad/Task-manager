# Task Manager App

A full-stack task management application that allows users to create, update, and delete tasks. The app helps users organize their work efficiently with a clean and responsive interface.

## 🚀 Tech Stack

Frontend
- React
- Tailwind CSS
- Axios

Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

Deployment
- Vercel (Frontend)
- Render (Backend)

---

## ✨ Features

- Create new tasks
- Edit existing tasks
- Delete tasks
- Persistent cloud database storage
- RESTful API for task operations
- Responsive user interface
